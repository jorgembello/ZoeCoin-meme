# 🦄 ZoeCoin – The MemeCoin of Emotions

ZoeCoin (ZOE) is a playful ERC-20 token that celebrates kindness, emotional intelligence, and meme energy.

## 🚀 What is ZoeCoin?

ZoeCoin is a lighthearted memecoin deployed on the Base Sepolia testnet — a symbol of fun, connection, and digital feels.

## 💖 Features

- ✅ 1,000,000 initial supply
- ✅ Built using OpenZeppelin's ERC20 standard
- ✅ Deployed on Base Sepolia testnet
- ✅ Free to use for testing, demos, and emotional experiments

## 🛠 Tech Stack

- Solidity ^0.8.0
- OpenZeppelin Contracts
- Base Sepolia
- Hardhat or Remix

## 📝 Deploy Instructions (Remix)

1. Open [Remix IDE](https://remix.ethereum.org)
2. Create a new file `ZoeCoin.sol` and paste the contract code.
3. Compile with `Solidity ^0.8.0`
4. Deploy with “Injected Provider - MetaMask” and select **Base Sepolia**
5. Confirm the transaction in MetaMask
